## Summary

Upgraded the frontend build tool from Vite 5.0.0 to Vite 7.3.1 in `cockpit/app/frontend/`.

## Changes Made

### 1. Package Version Updates
- **frontend/package.json**:
  - `vite`: `^5.0.0` → `7.3.1`
  - `lit`: `^3.1.0` → `^3.3.0` (dependency update during upgrade)

### 2. Build System Migration (npm → bun)
- **build/Taskfile.yml**: Updated all frontend tasks to use `bun` instead of `npm`:
  - `install:frontend:deps`: `npm install` → `bun install`
  - `build:frontend`: `npm run build -q` → `bun run build` (removed `-q` flag - incompatible with Vite 7)
  - `dev:frontend`: `npm run dev` → `bun run dev`
  - Lockfile reference: `package-lock.json` → `bun.lock`
  - Precondition check: `npm version` → `bun --version`

## Compatibility Verification

### Vite 7 Breaking Changes Checked
- **Sass legacy API**: Not used in this project
- **splitVendorChunkPlugin**: Not used
- **Hook-level enforce/transform**: Not used

- **Known issues**: Vite 7.1.8 had connectivity bug with Wails (#4620), but **fixed in Vite 7.1.9**
